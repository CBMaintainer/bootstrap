JTUX

Get the latest release

Designed for the legal industry, everywhere.

JTUX makes front-end web development for court software faster and easier. It's kept up to date with Bootstrap & Journal Technologies developers for devices of all shapes, and projects of all sizes.

Get Started

Using JTUX is just like using bootstrap but with added components for court software. If you have never used JTUX before, Get started now!

eCourt components

JTUX epands on the foundation Bootstrap has created by adding court components. eCourt is one of the core components to JTUX and journal technologies software.

jcon

JTUX comes with Bootstraps icon set within the build. jcon hosts cort specific icons that work seamlessly with JTUX. Get some jcon's!

Preprocessors

JTUX ships with vanilla CSS, but its source code utilizes the two most popular CSS preprocessors, Less and Sass. Quickly get started with precompiled CSS or build on the source.

One framework, every device.

JTUX easily and efficiently scales your websites and applications with a single code base, from phones to tablets to desktops with CSS media queries.

Full of features

With JTUX, you get extensive and beautiful documentation for common HTML elements, dozens of custom HTML and CSS components, and awesome jQuery plugins.

JTUX is open source. It's hosted, developed, and maintained on GitHub.
