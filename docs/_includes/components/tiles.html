Tiles

Provide a set of related clickable options that are easily glanceable and compareable.

Use Bootstrap's grid system to arrange the tiles with sensible sizing. Tiles always sit within a class="row tile-row"

Simple Tile

At it's simplest form, tiles show an abbreviation or icon and a title.
