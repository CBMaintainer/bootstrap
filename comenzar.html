Acerca de BAstrap3

BAstrap3 se usa para hacer el desarrollo front-end de los sitios web de GCBA.

Usa Bootstrap 3 y está compuesta por código HTML, CSS y JavaScript.

Para su uso es necesario contar con conocimientos de HTML, CSS y Bootstrap 3 (ya que la mayoría de los componentes y clases son los mismos).

Es importante tener en cuenta que los archivos NO se bajan. Se los llama desde las declaraciones del servidor oficial y luego el sitio se arma copiando y pegando el código de estilos y componentes.

Índice

La documentación de este sitio está compuesta por:

Ejemplos, declaraciones de archivos del servidor oficial y uso de la grilla multi-columna.

Elementos compuestos con funciones especiales. Algunos funcionando con JavaScript.

Guía de diseño web abarcando conceptos de estética, comunicación y función. UIkit para Illustrator y Photoshop, manual de marca, logos y otros recursos.

Código base

Descargando el siguiente archivo podés comenzar a usar BAstrap3.

Ver archivo HTML

Se debe respetar la siguiente estructura en el encabezado (tags "html" y "head") de las páginas del sitio:

  • Doctype HTML5
  • Atributo de lenguaje español
  • Codificación UTF-8
  • Compatibilidad con IE
  • Compatibilidad mobile
  • Descripción del sitio para buscadores
  • Autor del sitio
  • Título de la página
  • Íconos del sitio para diferentes dispositivos
  • Link de bootstrap minimizado en su CDN
  • Link de BAstrap3 minimizado en su CDN
  • Link de estilos extra del sitio
  • Compatibilidad condicional con IE comentada

Se debe respetar la siguiente estructura en el pie (antes del cierre del tag "body") de las páginas del sitio:

  • Link de jQuery minimizado en su CDN
  • Link de JavaScript de Bootstrap minimizado en su CDN
  • Scripts de inicialización de componentes usados en la página/sitio

También se deben respetar el encabezado y pie de página oficiales.

Grilla base

La grilla de BAstrap3 está compuesta principalmente por 3 elementos:

  • Contenedor (Container)
  • Fila (Row)
  • Columna (Col)

El sufijo "-fluid" sirve para que el ancho total se adapte al contenedor padre (que puede ser diferente al ancho total de la página).

<div class="container">
  <div class="row">
    <div class="col-md-12">
      ...
    </div>
  </div>
</div>

<div class="container-fluid">
  <div class="row-fluid">
    <div class="col-md-12">
      ...
    </div>
  </div>
</div>
