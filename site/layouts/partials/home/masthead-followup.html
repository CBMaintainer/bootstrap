التركيب

قم بتثبيت ملفات Sass و JavaScript المصدر الخاصة بـ Bootstrap عبر npm ،Composer، أو Meteor.

لا تتضمن عمليات التثبيت المُدارة للحزمة الوثائق أو البرامج النصية للبناء الكامل الخاصة بنا. يمكنك أيضًا استخدام نموذج npm repo الخاص بنا لإنشاء مشروع Bootstrap بسرعة عبر npm.

jsDelivr

عندما تحتاج فقط إلى تضمين CSS أو JS المترجم من Bootstrap، يمكنك استخدام jsDelivr.

شاهده عمليًا من خلال نموذج البداية البسيط الخاص بنا، أو تصفح الأمثلة لبدء مشروعك التالي.  يمكنك أيضًا اختيار تضمين Popper و JS الخاص بنا بشكل منفصل.

أيقونات Bootstrap

لأول مرة في التاريخ، يمتلك Bootstrap مكتبة أيقونات SVG مفتوحة المصدر خاصة به، مصممة للعمل بشكل أفضل مع المكونات والوثائق الخاصة بنا.

تم تصميم أيقونات Bootstrap لتعمل بشكل أفضل مع مكونات Bootstrap، لكنها ستعمل في أي مشروع. إنها ملفات SVG، لذا فهي تتوسع بسرعة وسهولة، ويمكن تنفيذها بعدة طرق، ويمكن تصميمها باستخدام CSS.
